## Instructions (Linux)
- `cd controller`
- `python -m venv venv`
- `source venv/bin/activate`
- `pip install -r requirements.txt`
- `python3 main.py`

## arguments
- `-h --help` Display a list of all commands
- `--verbose` Print logs
- `--window` Show a windows with cv2 output
