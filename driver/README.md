# DSDriver

## Controls

- `*IDN?` - Indetify device
- `LED` - Control the builtin LED
	- `:ON` - Toggle ON
	- `:OFF` - Toggle OFF
- `LOCK` - Control the lock (motor)
	- `:ON` - move until locked
	- `:UNLOCK` - move until unlocked
